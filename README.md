# Housing Helper NYC

Housing Helper NYC is a simple Next.js + TypeScript + Tailwind CSS web app that collects basic household information and shows filtered sample apartment listings for New York City.

## Features

- Homepage with a simple housing search form
- Results page that clearly shows submitted user inputs
- Sample listing data in `data/listings.ts`
- Filtering by borough preference and max rent
- Eligibility disclaimer message

## Setup

### 1) Install dependencies

```bash
npm install
```

### 2) Run the development server

```bash
npm run dev
```

### 3) Open the app

Visit [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm run start
```
